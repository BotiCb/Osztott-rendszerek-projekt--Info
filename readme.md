# Backend - NestJS  

### dokumentacio: https://docs.nestjs.com/

## 🚀 Indítási Útmutató  

```bash
cd api  
yarn install  
yarn start:dev  
```

## 📝 `.env.yml` fájl  
Másold be az **`.env.yml`** fájlt az **api** root mappájába. A fájlt a Messenger csoportban találod.
